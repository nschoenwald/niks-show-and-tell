# Changelog

## [13.01] - Initial Release

### Features
*   **Context Menu**: Added right-click menu to images in Journal Entries and Image Popouts.
    *   **Show**: Pop out the image locally.
    *   **Send to Chat**: Open a sharing dialog.
    *   **Copy URL**: Copy image source to clipboard.
*   **Seamless Sharing**:
    *   **Paste**: Paste images (Ctrl+V) directly into Foundry to share them.
    *   **Drag & Drop**: Drag images from your computer onto the Chat Log to upload and share.
    *   **Safe URL**: Pasting a direct image URL opens the sharing dialog instead of treating it as text.
*   **Smart Dialog**: The "Send to Chat" dialog features:
    *   **Thumbnail**: Preview the image before sending.
    *   **Caption**: Add custom flavor text to your image.
    *   **Privacy**: Choose to "Show to All" or "Whisper" to specific players.
*   **Optimization**:
    *   **WebP Compression**: Automatic client-side compression for uploads (configurable quality) to save server space.
*   **Accessibility**:
    *   **Scene Tools**: Added "Show Copied URL" tool in Tile Controls for mouse-only workflows.
